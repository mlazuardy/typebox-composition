# Using React

`Typebox Composition` has same behavior as `typebox` that it support both NodeJS and Browser / Client as well as our custom format.

> If you want to re-use `TypeComposition` instance for Both NodeJS and Browser, make sure your custom type or format support both environment.
Otherwise, you need to create each instance for each environment.
