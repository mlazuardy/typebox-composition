# Installation

## Adding Typebox Validator
Make sure you have an up-to-date NodeJS version (prefer 18++) installed on your local machine.

Start by installing our package and its peerDepedencies
```bash
npm install @sinclair/typebox @typeb/validator

yarn add @sinclair/typebox @typeb/validator

pnpm add @sinclair/typebox @typeb/validator
```

After that create `TypeValidator` instance
```typescript
import { TypeValidator } from "@typeb/validator";

export const validator = new TypeValidator({ lang: "en" });
```
